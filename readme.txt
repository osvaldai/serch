Предполагается что все файлы лежат в папке d:\88\
1. Скачиваем питон Python https://www.python.org/
2. Запускаем скачанный файл и ставим питон, не забываем отмечать галочку возле path
3. Устанавливаем библиотеку bit (нажимаем WIN+R, вводим следующую команду в строку, нажимаем enter): pip install bit
4. Качаем базу адресов с балансом https://gz.blockchair.com/bitcoin/addresses/
5. Скачанный файл распаковываем и обрабатываем с помощью create_base.py
6. Запускаем скрипт, ради которого все мучения: WIN+R python d:\88\start.py
